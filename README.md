# Incident Trend & Root Cause Analysis
An end-to-end data analytics project focused on analyzing IT incident data to uncover trends, identify root causes, and improve resolution efficiency. This project simulates a real-world incident management system and showcases practical data analysis skills.

### Tools & Technologies

* Python (Pandas, Data Cleaning, Feature Engineering)
* SQL (Data Transformation, Aggregation, Query Optimization)
* Power BI (Interactive Dashboard & Visualization)

### Project Workflow

* Cleaned and preprocessed raw event log data using Python
* Transformed data into incident-level dataset using SQL
* Engineered key metrics like resolution time (MTTR) and root cause mapping
* Built an interactive Power BI dashboard for real-time trend analysis

### Key Insights

* Identified high-frequency incident categories and recurring root causes
* Analyzed resolution performance across different teams
* Detected trends based on severity, time, and incident patterns

### Impact

* Enables faster identification of critical issues
* Helps improve incident response and operational efficiency
* Demonstrates real-world problem-solving using data analytics
